# WeAirSim eSIM

This repository contains the new WeAirSim eSIM web platform and cross-platform mobile app source code.

## Structure

- `esim-international-esim-and-data-purchase-platform/` - web/backend platform source and documentation.
- `esim-cross-platform-mobile-app-for-international-esim-and-data-purchase/` - Flutter Android/iOS app source and documentation.
- `google-play-launch-checklist.md` - current Google Play/Firebase launch checklist.
- `airalo-api-next-actions.md` - Airalo API onboarding next actions.
- `supplier-payment-migration-checklist.md` - supplier/payment migration checklist.

## Security

Runtime secrets, OAuth credentials, Firebase local config files, upload keystores, generated bundles, deployment archives, and local screenshots are intentionally excluded from Git.
